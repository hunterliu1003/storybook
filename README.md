## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:9000
$ npm run storybook

# build static html for storybook
$ npm run build:storybook

# create new component
$ npm run plop

# enter the component name (properCase)
# example: TheButton
```
